API

class verizon_connect_api.VerizonConnectAPI.VerizonConnectAPI(app_id: str, username: str, password: str, api_url='https://fim.api.us.fleetmatics.com:443/')

Bases: object

This module handles authentication and data-fetching with the Verizon Connect API. Currently, Verizon Connect uses the “Basic” HTTP authentication scheme, which requires a username and password. Use of the API will require registration of an app in their Developer Portal.

Parameters:

  • app_id (str) – ID of app registered in Verizon Connect Developer Portal

  • username (str) – Username for account generated during app registration

  • password (str) – Password for account generated during app registration

  • api_url (str) – API endpoint, defaults to ‘https://fim.api.us.fleetmatics.com:443/’
